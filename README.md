Healthy Minds project for Software Engineeting I course in UEF

## Current Status

The platform is currently connected to our own Supabase project and includes:
- authentication
- role handling
- dashboards
- Finnish population health data via FastAPI

Known issues:
- Patient invitation flow is currently incomplete and does not yet work at all.
- Email confirmation is disabled for now due to multiple issues with login. 
- Sotkanet integration has been moved to FastAPI and is working through the backend service.
- Further cleanup is still needed.
